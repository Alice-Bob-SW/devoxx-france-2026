# Devoxx France 2026 Demo - Alice & Bob

The Jupyter notebook with most of the outputs is available online 
[here](https://github.com/Alice-Bob-SW/devoxx-france-2026/blob/main/devoxx_demo.ipynb).

But to see all the content including videos, run it locally.

## Run Jupyter notebook

Requirements:
- [uv](https://docs.astral.sh/uv/)

Run :
```bash
uv sync --frozen
uv run --with jupyter jupyter lab devoxx_demo.ipynb
```
